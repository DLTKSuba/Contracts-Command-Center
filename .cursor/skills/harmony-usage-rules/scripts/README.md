# harmony-usage-rules scripts (extraction helpers)

The files in this folder (**`HEADING_PATTERNS.md`**, related notes in **`PLAN_UPDATES.md`**) describe **automation for extracting usage rules from the full Harmony documentation site or documentation package**—not from the Harmony Designer Starter file tree.

In **this kit**, prefer reading **`RULES.md`** from an installed Harmony npm package when present (for example under `docs/RULES.md` inside `@deltek/harmony-components` or `@deltek-sst/harmony-design-system`). Use the heading-pattern docs here only when you are maintaining or running rule-extraction tooling against upstream HTML/markdown sources.
