# Cursor skills (Harmony designer starter)

These skills ship with the designer starter kit. **All skills live under `.cursor/skills/`** in this project.

| Folder | Purpose |
|--------|---------|
| `harmony/` | Hub: components, ShellLayout, theme/mode, natural-language workflows. |
| `harmony-usage-rules/` | Compliance and accessibility; `RULES.md` when a Harmony package is installed. |
| `harmony-ux-principles/` | UX critique. |
| `design-patterns/` | Pattern registry, `reference/*.md`, `create_pattern.py`, `search_patterns.py`. |
| `layout-builder/` | Layout patterns and composition rules inside the React shell (`src/components/harmony/`). |
| `build-layout/` | Playbook: **`/build-layout`** (uses layout-builder). |
| `build-all-patterns/` | Playbook: **`/build-all-patterns`**. |
| `create-pattern/` | Playbook: **`/create-pattern`**. |
| `search-patterns/` | Playbook: **`/search-patterns`**. |
| `harmony-critique/` | Playbook: **`/harmony-critique`**. |
| `ux-review/` | Playbook: **`/ux-review`**. |

**Not included:** `harmony-converter` (this kit is React-only).

See also [`.cursor/DESIGNER_GUIDE.md`](../DESIGNER_GUIDE.md) for slash commands and workflows, and [`HARMONY_DESIGNER_HANDBOOK.md`](../../HARMONY_DESIGNER_HANDBOOK.md) for the full handbook.
